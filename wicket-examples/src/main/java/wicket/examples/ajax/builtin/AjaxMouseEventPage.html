Demonstrates the use of wicket.extensions.ajax.AjaxMouseEventBehavior which has support for getting the event properties for mouse based AJAX events on the server. This is values such as X and Y position and which button has been pressed.

Examples Debug output
Click on this box to execute an onClick javascript event.
Press any mouse button on this box to execute an onMouseDown javascript event.
Press and release any mouse button on this box to execute an onMouseUp javascript event.
Move the cursor over this box to execute an onMouseOver javascript event.
Move the cursor over and then out this box to execute an onMouseOut javascript event.
Move the cursor over this box to execute an onMouseMove javascript event.

(PLEASE NOTE!) This will generate many AJAX requests.
